# Cavapendolandia - Checklist Go/No-Go

## Gate A - Stabilizzazione
- [ ] `npm run lint` senza errori.
- [ ] `npm run build` completato.
- [ ] `npm run test` passante.
- [ ] `.env.example` presente e README aggiornato.

## Gate B - Sicurezza contenuti
- [ ] Upload senza consensi obbligatori bloccato.
- [ ] Link non `http/https` bloccato.
- [ ] File oltre 100 MB bloccato.
- [ ] Media `pending/rejected/hidden` non visibili pubblicamente.

## Gate C - Esperienza pubblica
- [ ] Home chiara in meno di 10 secondi.
- [ ] Archivio con `Vaga`, `Nuovi arrivi`, `Silenzio`.
- [ ] Flusso `offri` completabile in meno di 1 minuto.
- [ ] Tono lessicale coerente e non-social.

## Gate D - Operativita admin
- [ ] Queue admin filtrabile per media.
- [ ] Stato `hidden` gestibile da tab dedicata.
- [ ] Approva/Rifiuta/Oscura/Ripristina funzionanti.
- [ ] Procedura rimozione documentata.

## Gate E - Condivisione
- [ ] Seed curatoriale pronto (8-15 offerte).
- [ ] Kit review Antonio pronto.
- [ ] Nessuna visual derivativa da opere originali.
- [ ] URL live stabile.
